## О проекте

`Christmas Shop` — простая статическая витрина магазина подарков с рождественским оформлением.

## Запуск локально

Это статический набор HTML/CSS/JS. Для локального просмотра просто откройте `home.html` или `gifts.html` в браузере. Для разработки удобно поднять локальный сервер:
```bash
# Python 3
cd christmas-shop
python -m http.server 8000
# затем открыть http://localhost:8000/home.html
```
